Al enviar una petición POST a http://localhost:3000/registro con el JSON { "nombre": "Maria", "mensaje": "Hola comunidad" }, el servidor responde con un JSON que confirma la recepción de los datos. La respuesta devuelta es:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}
